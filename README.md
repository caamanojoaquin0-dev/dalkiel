<h1>Dalkiel</h1>
<p><strong>Dalkiel</strong> es un término de jerga chilena de uso coloquial utilizado de manera informal para referirse al pene, principalmente en contextos juveniles y humorísticos.</p>

<h2>Uso y contexto</h2>
<p>El término se emplea en situaciones informales, conversaciones privadas o expresiones humorísticas. No corresponde a un uso formal ni académico del idioma español.</p>

<h2>Registro cultural</h2>
<p>Este sitio web tiene como finalidad dejar constancia pública del uso del término <em>Dalkiel</em> dentro del habla coloquial en Chile.</p>
